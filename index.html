Overview

piDisplay is a 16x2 character LCD based minecraft server notification system. You will need a Minecraft server running this plugin and build the hardware described in this guide. For ease of testing, the plugin is built against Bukkit v12.1 as this is what ran best on the RaspberryPi 1 that I used for this. I also used Glowstone instead of Spigot for this same reason. There is however, no reason why it wouldn't run higher versions on a more recent RaspberryPi. It's therefore slower but at least I know that this will run on any Pi.

How to use

If you are not interested in playing with the code then just download the latest .jar file from the releases link. After downloading, copy it to your minecraft server plugins folder.
The plugin displays messages on the display whenever someone joins or leaves the server as well placing any blocks.

If you want to play with the source code then download the whole project from the links on the left. It was built in Eclipse using the Maven framework.

Hardware build

Schematic
The overall schematic, I used a 5V LCD which required level convertors for the 3.3V GPIO on the Pi. My life would have been easier if I'd just picked a 3.3V LCD in the first place. If you do this then you can just omit the level convertors.
Note that the resistor values were tuned to the actual LEDs/LCD that I had, YMMV. The original design that I did was based on using the full 8 bit interface so I could write entire bytes over the interface. This had to be shelved due to space constraints with the project box that I used but for the sake of completeness it's here

Stripboard plan
This is by no means of indication of how this should be done. I'd normally mark up a much neater layout. I made numerous errors (you wouldn't think I spent nigh on 20 years in industry doing PCB layout) mistaking which way round the boards and header cables were which I could only get round by refactoring the schematic connections and gpio pinouts in the software. To that end, consider the layout and schematic an as-built design at best.

PCBs
These are the actual boards. For the reasons above these should just be considered as an as-built record. Please ignore the copious evidence of refactoring the layout and poor soldering.

Box
I used a standard black plastic project box for this. A lower voltage LCD would enable the central level convertor board to be removed and a custom PCB should enable a single board design. I just used stripboard as it's what I had lying around.
At the top right of the photo you can see the contrast adjust potentiometer. I built it upside-down deliberately so the screws and lid would be hidden and it would look more seamless in the living room.
